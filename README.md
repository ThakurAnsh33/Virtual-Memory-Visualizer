🧠 Virtual Memory Management Visualizer

An interactive web-based tool to visualize key concepts of Operating Systems including Paging, Segmentation, and Memory Fragmentation.

🚀 Features
🔹 Paging & Page Replacement
Simulates LRU (Least Recently Used) and Optimal algorithms
Displays step-by-step execution
Shows page faults, hits, and hit ratio
🔹 Segmentation
Demonstrates logical to physical address translation
Uses Base and Limit registers
Validates memory access using offset
🔹 Fragmentation
Simulates First Fit memory allocation
Visualizes internal and external fragmentation
Displays remaining memory blocks and allocation status
💡 How It Works
User provides input (pages, frames, segments, memory blocks)
The system processes the data using implemented algorithms
Results are displayed in a structured and visual format

Understanding of Virtual Memory Management
Hands-on implementation of Page Replacement Algorithms
Clear visualization of Segmentation and Fragmentation concepts
